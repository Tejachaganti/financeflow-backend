# FinanceFlow Backend

FinanceFlow is a full-stack personal finance management application backend built using Node.js, Express.js, and MongoDB Atlas.

## Features

* User Authentication with JWT
* Expense Management APIs
* Budget Tracking APIs
* Analytics APIs
* Profile Management
* Secure REST API Architecture
* MongoDB Atlas Cloud Database
* Render Cloud Deployment

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* Render Deployment

## API Base URL

https://financeflow-backend-l0vt.onrender.com

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file and add:

```env
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLIENT_URL=*
```

## Deployment

* Backend deployed on Render
* Database hosted on MongoDB Atlas
